[3] TÂCHE — 4 ÉTAPES SÉQUENTIELLES

ÉTAPE 1 — PRODUCTION DES CODES
Depuis grille_v1 + fichier Excel ES joint :
  · 1 code comportemental par degré par critère
  · Format : 2 caractères MAX · lisible à 50cm sur fiche imprimée
  · Pour TYPE 1 (fourchette) : code + plage de notes
    Ex : A → XX− · 0 à 4 pts
         B → XX~ · 5 à 7 pts
         C → XX+ · 8 à 10 pts
  · Pour TYPE 2 (valeur chiffrable) : format strict
    Secondes → 10"2 · Minutes → 1'30" · Mètres → 0,90
  · Convention couleurs obligatoire :
    Degré A → fond #FCEBEB · Arial Bold ≥ 14pt
    Degré B → fond #FAEEDA · Arial Bold ≥ 14pt
    Degré C → fond #EAF3DE · Arial Bold ≥ 14pt

ÉTAPE 2 — ⚑ PAUSE INTERACTIVE ORIENTÉE
Après avoir produit le tableau des codes · STOP.
Présenter la recommandation de saisie selon [FAMILLE_APS] :

   → ENTOURER le niveau (A · B · C) [TYPE 1]

   Durée maximale recommandée : 3 secondes par élève.

     Pas de calcul mental · lecture directe
